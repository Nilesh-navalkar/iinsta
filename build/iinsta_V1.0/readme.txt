STEPS TO GET INSIGHTS :
=============================================================================================
1. edit keys.json --> add username and password --> save and close
2. run the "iinsta.exe"
3. wait until the chrome automatically closes  /// do no interfere the chrome tab in any way
4. insights will be created in a space separated file called "insights.txt"

* recomended to not run more than 5 times per day
